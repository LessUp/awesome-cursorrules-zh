# 演进思考

这页专门记录还没有被完全解决、但已经清晰浮现的演进问题。

## 演进问题一：快照层的粒度

当前快照层已经输出分类分布、覆盖桶和质量信号，但还可以继续研究：

- 是否补充更细粒度的规则质量信号
- 是否记录规则组合之间的关联强度
- 是否支持更强的跨页面数据复用

## 演进问题二：算法章节的可视化深度

规则编排算法已经可读，但还可以继续增强：

- 更丰富的升级路径图
- 更细的决策树分支
- 更强的场景化组合说明

## 演进问题三：竞品与证据库联动

未来可以把竞品分析和规则证据库做得更紧：

- 让竞品对照直接对应到规则组合模式
- 让参考文献与证据页形成更强的交叉引用
- 让审稿人更容易看到“本站到底多推进了一步”
