# 存储规则

存储相关的 Cursor AI 编程规则集合。

## dna-storage

| 规则 | 说明 |
|------|------|
| [dna-storage](https://github.com/LessUp/awesome-cursorrules-zh/blob/master/rules/storage/dna-storage/.cursorrules) | DNA存储 .cursorrules 提示文件 |

## 使用方法

1. 点击规则链接，在 GitHub 上查看 `.cursorrules` 文件
2. 复制内容到项目根目录的 `.cursorrules` 文件
3. 在 Cursor AI 中启用项目规则
4. 开始编码，AI 将遵循规则生成代码
