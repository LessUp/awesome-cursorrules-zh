# AR/VR Rules

Cursor AI programming rules for AR/VR.

## augmented-reality

| Rule | Description |
|------|------|
| [augmented-reality](https://github.com/LessUp/awesome-cursorrules-zh/blob/master/rules/ar-vr/augmented-reality/.cursorrules) | 增强现实开发 .cursorrules 提示文件 |

## Usage

1. Click the rule link to view the `.cursorrules` file on GitHub
2. Copy the content to the `.cursorrules` file in your project root
3. Enable project rules in Cursor AI
4. Start coding, AI will follow the rules to generate code
