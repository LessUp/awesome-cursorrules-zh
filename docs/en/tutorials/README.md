## Quick Start

### Step 1: Choose a Rule

Browse the [Rules Directory](https://awesome-cursorrules-zh.js.org/) to find rules suitable for your project.

### Step 2: Download the Rule

```bash
# Method 1: Direct copy
cp rules/frontend/react/nextjs-typescript/.cursorrules /your/project/

# Method 2: curl download
curl -o .cursorrules https://raw.githubusercontent.com/LessUp/awesome-cursorrules-zh/main/rules/frontend/react/nextjs-typescript/.cursorrules
```

### Step 3: Verify Activation

Open the project in Cursor editor, start coding, and AI will provide better suggestions based on `.cursorrules`.
