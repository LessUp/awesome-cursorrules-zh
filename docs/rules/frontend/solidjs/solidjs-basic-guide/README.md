# SolidJS 基础指南 .cursorrules 提示文件

## 你可以构建什么

- **高性能 Web 应用**：利用 SolidJS 的细粒度响应式系统
- **服务端渲染应用**：支持 SSR 的全栈应用
- **轻量级前端应用**：小巧包体积的高效应用

## 概要

这个规则文件为 SolidJS 项目提供了基础指导和最佳实践。它涵盖了响应式状态管理、副作用处理、条件渲染和性能优化等方面。

## 核心特性

- 使用 createSignal() 创建响应式状态
- 利用 createEffect() 处理副作用
- 实现 createMemo() 用于派生值
- 使用 createResource() 进行数据获取
- Show 和 For 组件

## 技术栈

- SolidJS
- JSX
- Solid Router
