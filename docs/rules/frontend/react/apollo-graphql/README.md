# Apollo GraphQL .cursorrules 提示文件

## 你可以构建什么

- **GraphQL 客户端应用**：使用 Apollo Client 与 GraphQL API 交互
- **实时数据应用**：支持订阅和实时更新的应用
- **离线优先应用**：支持本地缓存和离线操作的应用

## 概要

这个规则文件为使用 Apollo Client 的 React + TypeScript 项目提供了最佳实践指导。它涵盖了 Apollo Client 配置、缓存策略、自定义 Hooks 和数据变更等方面。

## 核心特性

- Apollo Client 配置和缓存管理
- 自定义 Hooks 封装查询和变更
- 缓存策略优化
- TypeScript 类型定义

## 技术栈

- React
- TypeScript
- Apollo Client
- GraphQL
