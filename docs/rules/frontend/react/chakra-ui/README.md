# Chakra UI .cursorrules 提示文件

## 你可以构建什么

- **企业级 React 应用**：快速构建美观的企业应用
- **可访问性优先的应用**：符合 WCAG 标准的无障碍应用
- **主题化应用**：支持亮色/暗色模式的应用
- **响应式 Web 应用**：适配多设备的应用

## 概要

这个规则文件为使用 Chakra UI 的 React + TypeScript 项目提供了最佳实践指导。它涵盖了组件组合、可访问性、暗模式和响应式设计等方面。

## 核心特性

- 利用 Chakra UI 组件库构建 UI
- 内置 WAI-ARIA 支持
- 支持亮色/暗色模式切换
- 响应式样式和布局

## 技术栈

- React
- TypeScript
- Chakra UI
