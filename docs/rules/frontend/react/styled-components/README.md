# Styled Components .cursorrules 提示文件

## 你可以构建什么

- **现代 React 应用**：使用 CSS-in-JS 构建样式隔离的组件库
- **主题化应用**：支持多主题切换的 Web 应用
- **组件库**：可复用的 UI 组件库
- **响应式布局**：基于 props 的动态样式系统

## 概要

这个规则文件为使用 Styled Components 的 React + TypeScript 项目提供了最佳实践指导。它涵盖了组件样式定义、条件样式、主题系统和 TypeScript 集成等方面。

## 核心特性

- 使用函数组件和 Hooks
- 支持 ThemeProvider 全局主题
- 创建可重用的样式化组件
- 使用 props 进行动态样式
- 完整的 TypeScript 类型支持

## 技术栈

- React
- TypeScript
- Styled Components
