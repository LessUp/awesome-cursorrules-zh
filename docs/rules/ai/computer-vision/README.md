# 计算机视觉规则集

> OpenCV、PyTorch、TensorFlow 视觉 AI 开发

## 技术栈

| 类别 | 技术 |
|------|------|
| 框架 | OpenCV, PyTorch, TensorFlow |
| 任务 | 目标检测、图像分割、图像分类 |
| 部署 | ONNX, TensorRT, 模型量化 |

## 核心规则

### 数据准备

- 规范化数据标注流程
- 实现数据增强策略
- 建立数据质量检查机制

### 模型开发

- 平衡精度与推理速度
- 选择合适的预训练模型
- 实现模型可解释性

### 部署优化

- 模型量化和剪枝
- 边缘设备适配
- 推理加速技术

### 评估标准

- 统一测试基准
- 多指标综合评估
- 建立模型对比框架

## 应用场景

- 工业质检系统
- 医学影像分析
- 自动驾驶感知
- 安防监控系统
- 人脸识别应用

## 使用方法

```bash
cp rules/ai/computer-vision/.cursorrules /你的项目/
```
