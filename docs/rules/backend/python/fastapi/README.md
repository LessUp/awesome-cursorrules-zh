# FastAPI 开发规范 .cursorrules 提示文件

## 你可以构建什么

- **高性能 API**：使用 FastAPI 构建快速、异步的 REST API
- **微服务**：可扩展的 Python 微服务架构
- **数据验证 API**：使用 Pydantic 进行严格数据验证的 API

## 概要

这个规则文件为 FastAPI 项目提供了全面的开发规范。它涵盖了核心原则、错误处理、依赖项和 FastAPI 特定指南等方面。

## 核心特性

- 函数式、声明式编程风格
- Pydantic v2 数据验证
- 异步数据库操作
- RORO 模式（接收对象，返回对象）
- 类型提示优先

## 技术栈

- Python
- FastAPI
- Pydantic v2
- SQLAlchemy 2.0 / asyncpg
