# React Native Expo .cursorrules 提示文件

## 你可以构建什么

- **跨平台移动应用**：一套代码同时运行在 iOS 和 Android
- **快速原型开发**：利用 Expo 快速构建和迭代
- **OTA 更新应用**：支持无线更新的移动应用

## 概要

这个规则文件为使用 Expo 的 React Native 项目提供了最佳实践指导。它涵盖了项目结构、导航、样式和性能优化等方面。

## 核心特性

- 使用函数式组件和 Hooks
- Expo SDK 功能和 API
- React Navigation 导航
- Expo 推送通知
- 离线支持

## 技术栈

- React Native
- Expo SDK
- TypeScript
- React Navigation
