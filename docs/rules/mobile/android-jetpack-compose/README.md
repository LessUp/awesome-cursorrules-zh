# Android Jetpack Compose .cursorrules 提示文件

## 你可以构建什么

- **现代 Android 应用**：使用声明式 UI 构建原生 Android 应用
- **Material Design 应用**：遵循 Material Design 3 指南的应用
- **可测试的应用**：采用清晰架构、易于测试的应用

## 概要

这个规则文件为使用 Jetpack Compose 的 Android 项目提供了最佳实践指导。它涵盖了项目架构、Compose UI 指南、测试和性能优化等方面。

## 核心特性

- 清晰架构（Domain、Data、Presentation 层）
- Material Design 3 组件
- Kotlin 协程和 Flow
- Hilt 依赖注入
- 单向数据流

## 技术栈

- Kotlin
- Jetpack Compose
- Material Design 3
- Hilt
- Coroutines / Flow
