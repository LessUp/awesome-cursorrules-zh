# CI/CD 流水线 .cursorrules 提示文件

## 你可以构建什么

- **自动化部署流水线**：从代码提交到生产部署的全自动化流程
- **多环境管理**：dev/stage/prod 环境的统一管理
- **安全 CI/CD**：凭证管理、权限控制的安全流水线

## 概要

这个规则文件为 CI/CD 流水线设计提供了最佳实践指导。它涵盖了流水线即代码、多环境支持、安全防护和测试策略等方面。

## 核心特性

- 流水线配置版本化
- 多环境支持（dev/stage/prod）
- 安全凭证管理
- 自动化测试策略
- 回滚机制

## 技术栈

- GitHub Actions
- GitLab CI/CD
- Jenkins
- Argo CD
