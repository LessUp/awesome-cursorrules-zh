# 零信任安全规则集

> 零信任架构、身份认证、微服务安全

## 技术栈

| 类别 | 技术 |
|------|------|
| 身份 | SPIFFE/SPIRE |
| 策略 | OpenPolicyAgent |
| 加密 | mTLS |
| 网格 | 服务网格安全 |

## 核心规则

### 身份认证

- 实现 mTLS 双向认证
- 统一身份管理
- 动态证书轮换

### 访问控制

- 基于角色的访问控制 (RBAC)
- 最小权限原则
- 动态访问策略

### 持续验证

- 实时信任评估
- 行为异常检测
- 自适应认证

### 数据安全

- 传输加密
- 存储加密
- 密钥管理

## 应用场景

- 零信任网络架构
- 微服务安全通信
- 多云环境安全
- 远程办公安全

## 使用方法

```bash
cp rules/security/zero-trust/.cursorrules /你的项目/
```
