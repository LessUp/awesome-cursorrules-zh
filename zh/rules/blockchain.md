# 区块链规则

区块链开发相关的 Cursor AI 编程规则集合。

## 智能合约

| 规则 | 说明 |
|------|------|
| [foundry](/rules/blockchain/solidity/foundry/) | Solidity Foundry 开发规范 |
| [smart-contract-security](/rules/blockchain/smart-contract-security/) | 智能合约安全规范 |

## Web3

| 规则 | 说明 |
|------|------|
| [web3](/rules/blockchain/web3/) | Web3 开发规范 |

## 高级协议

| 规则 | 说明 |
|------|------|
| [advanced-protocols](/rules/blockchain/advanced-protocols/) | 区块链高级协议规范 |

## 使用方法

1. 选择适合你项目的规则
2. 将 `.cursorrules` 文件复制到项目根目录
3. 在 Cursor AI 中启用项目规则
4. 开始编码，AI 将遵循规则生成代码
