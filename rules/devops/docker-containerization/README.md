# Docker 容器化 .cursorrules 提示文件

## 你可以构建什么

- **容器化应用**：将应用打包为可移植的 Docker 镜像
- **多容器应用**：使用 Docker Compose 编排复杂应用
- **云原生应用**：适用于 Kubernetes 部署的容器化应用

## 概要

这个规则文件为 Docker 容器化项目提供了最佳实践指导。它涵盖了镜像优化、安全实践、配置管理和健康检查等方面。

## 核心特性

- 多阶段构建优化镜像体积
- 非 root 用户安全运行
- 环境变量与配置分离
- 标准化日志输出
- 容器健康检查

## 技术栈

- Docker
- Docker Compose
- 容器安全扫描工具
