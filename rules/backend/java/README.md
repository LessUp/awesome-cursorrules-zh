# Java 后端开发 .cursorrules 提示文件

## 你可以构建什么

- **企业级后端服务**：使用 Spring Boot 构建稳健的后端
- **微服务架构**：可扩展的 Spring Cloud 微服务系统
- **数据驱动应用**：使用 JPA 进行数据持久化的应用

## 概要

这个规则文件为 Java 后端项目提供了全面的开发规范。它涵盖了分层架构、实体设计、服务模式和异常处理等方面。

## 核心特性

- 分层架构（Controller-Service-Repository）
- Spring Boot 3 最佳实践
- JPA 实体设计
- 异常处理机制
- Lombok 代码简化

## 技术栈

- Java 17
- Spring Boot 3
- Spring Data JPA
- PostgreSQL
- Lombok
